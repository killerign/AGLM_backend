Pull the repository and follow these steps:

1) Install node js if not present : https://nodejs.org/en/download/

2) Create a new terminal (ctrl+shift+`) in VScode and navigate to the project folder.

3)Run npm install . in the terminal

4)Go to package.json and click debug above start field in the json.

Finally, use Postman for testing with the help of this documentation : https://docs.google.com/document/d/1QQ319oMc588JS4avO3QU_pEmXKCqtC_wQtwQnwVZ4dU/edit?usp=sharing
